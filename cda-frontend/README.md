# CDA Frontend

Interface React do Ciclo de Desempenho Automatizado.

## Como rodar

```bash
cd cda-frontend
npm install
cp .env.example .env
npm run dev
```

A interface sobe normalmente em `http://localhost:3000` ou na porta indicada pelo Rsbuild.

## Scripts

```bash
npm run dev      # desenvolvimento
npm run check    # valida TypeScript
npm run build    # gera build de produção
npm run preview  # prévia da build
```
